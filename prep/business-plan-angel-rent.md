# angel.rent - Business Plan & Technical Specification
## Boutique Hospitality Platform for El Ángel, Reforma 326, CDMX

---

## Executive Summary

**angel.rent** is a cutting-edge web application serving as the digital concierge and community hub for El Ángel - a boutique short-term rental property on Reforma 326, CDMX. The platform blends transactional efficiency with experiential luxury, creating a Soho House-inspired digital ecosystem that extends the boutique experience beyond physical walls.

**Investment Context:** Supporting a $1.5M MXN annual investment project with guaranteed monthly returns of $137,500 MXN, targeting 50-80% occupancy at $10,000-$16,000 MXN per night.

**Target Market:** Premium digital nomads, international creatives, expatriates, and luxury travelers attending major CDMX events (Mundial 2026, F1, Art Week, Día de Muertos).

## 3. Technical Architecture

### 3.1 Frontend Stack

**Framework:** React 18+ with TypeScript
- Component library: Custom components + shadcn/ui for consistency
- State management: React Context API + Zustand for complex state
- Routing: React Router v6
- Form handling: React Hook Form + Zod validation

**Styling:** Tailwind CSS
- Custom design system tokens
- Dark mode support
- Responsive breakpoints: mobile (320px), tablet (768px), desktop (1024px), wide (1440px)

**Progressive Web App (PWA)**
- Offline functionality for essential guest information
- Installable on iOS/Android home screens
- Push notifications for booking confirmations and guest messages
- Background sync for photo uploads

### 3.2 Backend Stack

**API Framework:** Next.js 14+ (App Router)
- Server-side rendering for SEO
- API routes for backend functionality
- Image optimization built-in
- Edge runtime for global performance

**Database:** PostgreSQL 15+
- Prisma ORM for type-safe database access
- Tables: users, bookings, properties, reviews, messages, tokens
- Full-text search for neighborhood guide
- Geospatial queries for location-based features

**Caching Layer:** Redis
- Session management
- API response caching
- Rate limiting
- Real-time features (occupancy updates)

**File Storage:** Cloudinary
- Image optimization and transformation
- Video hosting for property tours
- CDN delivery for global performance
- Automatic format conversion (WebP, AVIF)

### 3.3 Key Integrations

**Payment Processing:** Stripe
- Multi-currency support (MXN, USD, EUR)
- Automatic tax calculation for Mexican guests
- Subscription billing for future loyalty tiers
- Refund and dispute management

**Communication:**
- Twilio: SMS notifications for booking confirmations and check-in codes
- SendGrid: Transactional emails (confirmations, reminders, receipts)
- WhatsApp Business API: 24/7 guest support chat

**Calendar Management:**
- iCal sync with Airbnb, VRBO, Booking.com
- Google Calendar API for admin coordination
- Automated blocking for cleaning days

**Maps & Location:**
- Google Maps API: Neighborhood guide, directions
- Mapbox: Custom styled maps matching brand aesthetic
- Geolocation API: Distance calculations, nearby recommendations

**Analytics:**
- Plausible Analytics: Privacy-focused, GDPR-compliant
- Custom event tracking for conversion funnel
- Admin dashboard metrics

**Content Management:**
- Sanity.io: Flexible CMS for neighborhood guide, blog posts
- Real-time collaboration for Abril's team
- Versioning and scheduling

### 3.4 Security & Compliance

**Authentication:** NextAuth.js
- Email/password login
- OAuth providers (Google, Apple) for frictionless signup
- Two-factor authentication for admin accounts
- Session management with JWT

**Data Protection:**
- SSL/TLS encryption for all data transmission
- Database encryption at rest
- PII data anonymization in analytics
- Regular security audits

**Privacy Compliance:**
- GDPR compliance for European guests
- CCPA compliance for California guests
- Mexican data protection (LFPDPPP) compliance
- Cookie consent management
- Data export and deletion tools

**Payment Security:**
- PCI DSS compliance via Stripe

## 8. Risk Mitigation Strategies

### 8.1 Occupancy Risk
**Challenge:** Not meeting 50% base occupancy projection

**Mitigation:**
- Diversified marketing across OTAs + direct + creative network
- Dynamic pricing to fill low-demand periods
- Cláusula de amortiguación provides $40,000 MXN monthly buffer
- Minimum stay requirements during peak season maximize revenue
- Strategic partnerships with corporate housing, relocation services

### 8.2 Technology Risk
**Challenge:** Platform downtime or bugs affecting bookings

**Mitigation:**
- 99.9% uptime SLA with hosting provider
- Automated monitoring and alerting
- Manual booking backup process (WhatsApp, email)
- Regular backups and disaster recovery plan
- Staging environment for testing before production deployment

### 8.3 Competition Risk
**Challenge:** Similar luxury rentals competing for same guests

**Mitigation:**
- Unique creative community positioning (hard to replicate)
- Superior technology platform
- Abril's personal brand and network
- Content marketing showcasing property uniqueness
- Loyalty program creates switching costs

### 8.4 Regulatory Risk
**Challenge:** CDMX short-term rental regulations changing

**Mitigation:**
- Legal review of compliance requirements
- Register property properly with authorities
- Build relationships with building management
- Monitor regulatory changes proactively
- Diversify to corporate housing if needed (30+ day stays exempt from many restrictions)

### 8.5 Reputational Risk
**Challenge:** Negative reviews or guest incidents damaging brand

**Mitigation:**
- Rigorous guest screening and ID verification
- 24/7 support for rapid issue resolution
- Proactive communication and expectation setting
- Review response protocol (respond to all within 24 hours)
- Property insurance and liability coverage

### 8.6 Financial Risk
**Challenge:** Operating costs exceeding $15,000 MXN monthly budget

**Mitigation:**
- Detailed expense tracking and budget monitoring
- Vendor relationships with negotiated rates
- Automation to reduce labor costs (self check-in, smart home)
- Regular financial review with Abril
- Contingency fund for unexpected repairs
